# 🏆 SOLID Status Now

✅ SRP — Single Responsibility Principle

```text
One class = One responsibility
```

Example:

* LoginPage
* DriverFactory
* LoginTest

---

✅ OCP — Open Closed Principle

```text
Open for extension
Closed for modification
```

Example:

* Role interface
* AdminRole
* ManagerRole
* UserRole

---

✅ LSP — Liskov Substitution Principle

```text
Child should safely replace Parent
```

Example:

* Duck ✅
* Penguin ❌
